번역 작업
전체 번역 마이그레이션
모든 텍스트 데이터를 GPT를 사용하여 영어, 일본어, 중국어, 스페인어로 번역합니다.
* 이 작업은 API 비용이 발생할 수 있습니다.
자동 번역
데이터가 추가되거나 수정될 때 자동으로 번역이 수행됩니다.
* 비동기로 처리되어 저장 후 잠시 후 번역이 완료됩니다.
사용 안내
다국어 시스템 작동 방식
  1. 자동 번역: 서비스, 옵션, Fade Text 등을 추가/수정하면 GPT가 자동으로 4개 언어로 번역합니다.
  2. 언어 선택: 사용자가 웹사이트에서 언어를 선택하면 해당 언어의 번역된 텍스트가 표시됩니다.
  3. 번역 저장: 번역된 텍스트는 MongoDB에 저장되어 빠르게 조회됩니다.
  4. 폴백: 번역이 없는 경우 원본 한국어 텍스트가 표시됩니다.
